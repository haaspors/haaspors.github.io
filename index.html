About

I'm a software engineer mostly hacking on open source software written in compiled languages. (Read: C!) I'm one of those people who likes fiddling with bits and bytes instead of coming up with abstractions or patterns in the form of AbstractSingletonProxyFactoryBean You'll most likely find me typing into my full-screen terminal. Either in iTerm2 or terminator, using tmux+vim. My preferred shell is zsh using Oh My Zsh for configuration convenience.

I started programming in the '90s using BASIC/Visual Basic and C/C++ on MS-DOS and Windows. I spent my high school years hacking on childish Visual Basic apps before going to a Scandinavian university college to study C/C++, compilers, algorithms and computer science in general.

These days I'm first and foremost a dad for a wonderful litle boy, secondly I try to get some time to hack on computer science related projects, both at work and in in my spare time. You'll find some of my stuff at github obviously.
